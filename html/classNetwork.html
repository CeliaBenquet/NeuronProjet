Detailed Description

Set of Neurons connected and interracting together.

Each neuron receive a certain number of connections and sends a potential to all its connected neurons

void Network::createConnexions ( )

Create the connexions_ table of each neuron.

A connection is from an emitter neuron to the current instance. Those connexions are represented by random numbers. One number corresponds to the index of a neuron in the neurons_ table. The connexions_ table is constituted of connexions with exitatory then inhibitory neurons.
